# linbeam
LinBeam is a Python tool used for installing the program Bluebeam Revu 21 onto Linux using Wine.
## Roadmap
The end goal for this, is to have a wine-wrapper installer that will install and configure a Wine Prefix and setup Bluebeam Revu 21. 
This will be accomplished by
* System Dependancy Checks
* Prefix Construction
* A Winetricks Payload
* Wine-Wrapped MSI Installer
* Desktop Launcher for KDE

This will all be done through Python and a bit of Bash and all be wrapped up in a simple Bash executable.
